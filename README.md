# Swasthya Sathi – Mobile App UI

This repository contains the **front-end code** for **Swasthya Sathi**, a mobile-first web app UI designed for health and wellness services. It showcases a clean, responsive layout built with modern web technologies.

---

### Features
- Mobile-first responsive design  
- Smooth navigation between key sections (Login, Signup, Dashboard)  
- Lightweight and dependency-free UI using **Tailwind CSS** and vanilla JavaScript  
- Inline **SVG icons** for crisp, scalable visuals  
- Fully **offline-friendly** — just open `index.html` in a browser  

> This is the initial UI setup, with the core structure and navigation in place. Additional features and pages will be added as the project evolves.

---

### Tools & Technologies
| Tool/Tech | Purpose |
|-----------|--------|
| **HTML** | Structure and content of the web app |
| **Tailwind CSS (via CDN)** | Utility-first styling for responsive design |
| **JavaScript** | Simple navigation and interactivity |
| **Inline SVG Icons** | Scalable and clean iconography |

---

### Code Structure
- `index.html` – Complete UI layout, including login, signup, and dashboard sections  
- All styles included via Tailwind CDN  
- All icons embedded as inline SVGs for simplicity and scalability
