This repository contains the front-end code for **Swasthya Sathi**, a mobile-first web app UI designed for delivering health and wellness services. The app emphasizes a clean, responsive design and is built with modern web technologies to ensure a seamless experience across devices.

### Features
1. **Mobile-first responsive design**: Optimized for mobile users with smooth responsiveness on different screen sizes.
2. **User Authentication**: Includes Login and Signup pages for user authentication.
3. **Personalized Dashboard**: A user-friendly dashboard that displays key health information.
4. **Mood Check-in**: Users can track and log their mood to monitor mental health.
5. **Community Health Mapping**: Shows a map of health facilities and clinics in the community.
6. **Health Worker Directory**: A directory to connect users with available health workers.
7. **Seasonal Health Tips**: Offers health tips tailored to different seasons.
8. **Maternal and Child Health Tracking**: Tools to monitor maternal and child health.
9. **Anonymous Help Request**: Users can submit health-related requests confidentially.
10. **Voice-first Assistant**: A voice assistant that facilitates hands-free interaction with the app.

### Code Structure
- **index.html**: Contains the complete UI layout, including the login, signup, and dashboard sections.
- **Tailwind CSS**: Styles are included via the Tailwind CDN for a lightweight, responsive design.
- **Inline SVG Icons**: Icons are embedded as inline SVGs for improved performance and scalability.
